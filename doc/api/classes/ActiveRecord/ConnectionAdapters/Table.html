Class ActiveRecord::ConnectionAdapters::Table

  1. vendor/rails/activerecord/lib/active_record/connection_adapters/abstract/schema_definitions.rb
Parent: Object

Represents a SQL table in an abstract way for updating a table. Also see TableDefinition and SchemaStatements#create_table

Available transformations are: 

change_table :table do |t|
  t.column
  t.index
  t.timestamps
  t.change
